Projects


Moldelrithm - Github Link

Python library/framework for machine learning.

Takes an input of split training and testing data, and outputs the results of the accuracy, precision, and f-beta scores of 7 different Machine Learning algorithms, to determine which is the best for your data.

Tested on a Earth/Mars Classification problem, all parameters increased 2x (accuracy, precision, f-beta score)

Presented at/Created for Santa Barbara Hacks 2017 - Winner: Lockheed Martin aerospace challenge (due to use of satellite images for earth vs mars classification)


Udest - Github Link

Python, PHP, SQL - based web app that used IBM's Watson for machine learning, Amadeus' travel APIs to analyze text for emotive keywords and recommend the ideal US vacation spot, and the best deals on transport and hotels. Built for Calhacks Hackathon 2016

Collaborated with two other teammates using Git.

No longer functional due to discontinued access to IBM's Watson


AutoCV - Github Link

Python-based tool for creating resumes from given information on work experience.

Hoping to create a full neural network for total automation of resumes, as there are often many grammar, spelling, and translation errors in foreign candidate resumes.
